This Streamlit application extracts line-item data from Cadre Wire quote PDFs
and converts them into a single Excel spreadsheet.

## How to Run on Streamlit Cloud

1. Upload this repository to GitHub
2. Go to https://streamlit.io/cloud
3. Create a new app
4. Select:
   - Repo: cadre-quote-extractor
   - Main file: streamlit_app.py
5. Deploy

## Files

- streamlit_app.py : Main Streamlit app
- requirements.txt : Python dependencies
- index.html       : Reference HTML version (not executed by Streamlit)
